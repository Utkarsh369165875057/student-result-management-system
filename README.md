# 🎓 Student Result Management System

A Python-based desktop application to manage student records and results efficiently.  
This system allows adding, updating, deleting, and viewing student data with an easy-to-use interface.

---

## 📌 Features
- 👨‍🎓 Add new student records
- 📝 Update student details
- ❌ Delete student data
- 📊 Manage and view student results
- 🔍 Search student by Roll No / Name
- 💾 Database integration for data storage
- 🖥️ Simple and user-friendly GUI

---

## 🛠️ Tech Stack
- Python 🐍
- Tkinter (GUI)
- MySQL / SQLite (Database)
- Pillow (for images)
